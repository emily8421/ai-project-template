# 05 技术方案

## 1. 技术栈与版本
- Python 3.11
- 标准库 `argparse`
- 标准库 `pathlib`

## 2. 关键技术决策
- 命令行而非 HTTP：需求是本地文件处理，不需要服务化部署
- 同步文本处理：单文件、短文本场景下实现最简单
- 无持久化：每次执行只处理输入和输出文件，不记录历史状态

## 3. Phase 技术约束
与 `ai/project-rules.md` §1、§2 一致：当前 Phase 允许本地 CLI、标准库实现、同步执行；禁止数据库、Web API、图形界面和第三方 CLI 框架。
