# 01 用户需求说明

> 从愿景抽取完整需求，当前阶段只写细 Phase1。

- [P1][P1-已设计] 用户希望输入一个会议纪要文本文件。
- [P1][P1-已设计] 用户希望输出一个清理后的文本文件。
- [P1][P1-已设计] 用户希望去掉每行首尾空白。
- [P1][P1-已设计] 用户希望合并连续空行为单个空行。
- [P1][P1-已设计] 用户希望把 `*` / `+` 项目符号统一为 `-`。
- [P1][P1-已设计] 用户希望输入文件不存在或参数错误时得到清楚提示。
- [P2][骨架] 用户希望支持更多清理规则。
- [P2][骨架] 用户希望自动提取行动项。
- [愿景][骨架] 用户希望未来可导出更结构化的 Markdown。
