# 03 产品需求文档（PRD）

## 1. 功能范围
- 命令行入口与参数解析 —— 对应 REQ-1，当前 Phase1 内
- 文本基础清洗 —— 对应 REQ-2，当前 Phase1 内
- 输出文件写入 —— 对应 REQ-3，当前 Phase1 内
- 参数/文件错误提示 —— 对应 REQ-4，当前 Phase1 内

## 2. 优先级
- P0：命令执行、读取输入、清洗文本、写出结果、错误退出

## 3. Phase 划分
- Phase1（当前）：单文件文本清洗 CLI
- Phase2（预告）：支持可选规则组合、批量处理多个文件

## 4. 非目标（明确不做）
- 不做数据库或历史记录
- 不做 HTTP API 或前端界面
- 不做目录级批处理、异步任务、插件系统
