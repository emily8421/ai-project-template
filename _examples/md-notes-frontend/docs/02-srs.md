# 02 软件需求规格（SRS）

> Phase1 需求，每条可验证。

| ID | 需求 | 验证方式 |
|---|---|---|
| REQ-1 | 可创建新笔记，含标题与 Markdown 正文 | 新建后列表出现新条目 |
| REQ-2 | 编辑区输入 Markdown 时，预览区实时渲染 | 输入 `# 标题` 预览显示 h1 |
| REQ-3 | 笔记内容变更后自动写入 localStorage | 编辑后刷新页面，内容仍在 |
| REQ-4 | 列表展示所有已存笔记的标题 | 创建 N 条后列表显示 N 项 |
| REQ-5 | 可删除笔记，删除后从列表与存储中消失 | 删除后刷新页面，条目不再出现 |
| REQ-6 | 关闭浏览器后重开，笔记完整保留 | 写入 → 关闭 → 重开 → 内容一致 |

非功能：单页应用，零依赖，纯前端，离线可用。
