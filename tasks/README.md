# tasks/ 说明

Phase1阶段默认不启用本目录——`docs/08-dev-plan.md` 中每个Sprint按
`ai/global-rules.md` 第3节的格式书写即可。

当某个Sprint复杂到需要拆成多个独立任务时，将对应Sprint内容原样复制为
`task-00X-xxx.md`，格式如下：

```markdown

| 项 | 内容 |
```
